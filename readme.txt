Increase page speed adding display=swap to Google Fonts

== Description ==
Increase page speed and ensure text remains visible during Google webfont loading.
The plugin adds display=swap parameter to the google fonts url. 
	
This plugin works only with google fonts loaded from the Google Fonts directories. It does not work for local fonts.

== Installation ==
Just install the plugin and active it.
